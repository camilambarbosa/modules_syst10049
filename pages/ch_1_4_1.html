Content Models

A content model is a description of the element’s expected contents. An HTML element must have contents that match the requirements described in the element’s content model. The contents of an element are its children in the DOM (flow, interactive, embedded, phrasing, meta-data, sectioning, heading).

Block and inline flow

  • By default, block-level elements begin on new lines. Read more MDN web docs
  • By default, inline elements stay in line of the parent element. MDN web docs
  • To determine what content category a particular element belongs to, examine the specifications document.
