Accessibility

W3C develops international Web standards: HTML, CSS, and many more. W3C’s Web standards are called W3C Recommendations. All W3C standards are reviewed for accessibility support and their guidelines show how web accessibility depends on several components of web development and interaction working together, and how the WAI guidelines apply.

Read further W3C WAI Accessiblity Standards Overview

Accessibility principles

W3C WAI provides information about web accessiblity requirements, including

  • Web accessibility standards
  • Perceivable information and user interface
  • Operable user interface and navigation
  • Understandable information and user interface
  • Robust content and reliable interpretation
